## Major Highlights

- Fully Responsive
- Dark Mode
- Minimal Interface
- User friendly design
- Optimized for better performance
- Made using tAILWIND CSS.
- Single Quiz Page.
- 10 Question per Quiz category.
- Authentication (Signup, Login, and Logout)
